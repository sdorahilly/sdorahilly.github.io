Recent Work

DEA Map

This was a fictional experiment using raster data to determine "Bust" locations for a law enforcement operation in Missouri. Challenges for this project included data analysis and coming up with a layout that fit many requirements.

African Elephants

This project's goal was to experiment with table joins and relates. I also experimented with geoproccessing tools like union and intersect. The final result is a map that outlines areas of threat for elephants in several East African Countries.

GPS Map

This map was a short experiment with GPS Data. I collected the point, line, and polygon data using a GPS reciever on campus. I then created a simple map outlining a path to the Business School on my campus.

Farm Density Map

This is one of a series of maps created to demonstrate a consistent theme or map layout using similar data sets. These maps were all created using data I acquired using agricultural census data. This map is a comparison of using different methods of classification breaks.

Farm Dot Density Map

Mean Center
